# Spellcheck

Supervertaler can highlight misspellings while you translate.

## How it works

- Misspelled words are underlined in the grid.
- Right-click a misspelled word to see suggestions.
- You can **Add to Dictionary** or **Ignore** words.

## Language

Spellcheck is typically set to your project’s **target language**.

If spellcheck isn’t working, check:

- The selected spellcheck language
- Whether a Hunspell dictionary is installed (especially on Linux)

{% hint style="info" %}
If you’re on Linux and see instability, try disabling spellcheck temporarily and see [Linux-Specific Issues](../troubleshooting/linux.md).
{% endhint %}
