# Supported LLM Providers

Supervertaler supports multiple AI providers so you can choose what fits your workflow.

## Cloud providers

- **OpenAI** (GPT models)
- **Anthropic** (Claude models)
- **Google** (Gemini models)

## Local providers

- **Ollama** (run models locally, no API key)

## Choosing a provider

- If you want the easiest setup: start with one cloud provider.
- If you want privacy/offline: use [Ollama](ollama.md).

See also: [Setting Up API Keys](../get-started/api-keys.md)
