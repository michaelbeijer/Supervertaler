# Glossary of Supervertaler Terminology

## UI Elements

- Compare boxes
- Preview: the document preview
- Project editor, Project grid, Grid
- Termview: shows glosary matches in same layout as source text
- Translation results panel, Results panel (change to: Match panel?)
- focus border: the blue line around the selected target box
- Compare box
- TM Info
- Term Info

## Supervertaler tools

- **Functional**
  - AutoFingers (Automated translation pasting for memoQ from TMX files)
  - Superbrowser (Multi-Chat AI Browser)
  - Superbench (LLM Translation Quality Benchmarking System)
  - Superimage (Image Extractor)
  - Text Repair Tool
- **Needs work**
  - Supercleaner (DOCX tag cleaner)
  - Supervoice (Dictation system)
  - PDF Rescue (AI OCR)
  - TMX Editor


The "Keep Active Segment Centered" setting (similar to the memoQ/Trados setting) seems to be forgotten between restarts. That is, it doesn't seem to be saved to the project file.
