Features

Everything you need for professional translation work

📚

Okapi Framework File Filters

Battle-tested Okapi Framework localization filters run as an automatic background sidecar. Full round-trip DOCX with exact formatting preservation.

  • Full round-trip DOCX import and export
  • Semantic formatting tags (<b>, <i>, <u>)
  • SRX segmentation and proper paragraph detection
  • More formats coming: XLSX, PPTX, HTML, IDML
🤖

Multiple AI Providers

Choose from the best AI models for your translation needs.

  • OpenAI GPT-4o & GPT-4 Turbo
  • Anthropic Claude Opus, Sonnet, Haiku
  • Google Gemini 2.5 Pro/Flash
  • Local models via Ollama
💾

Translation Memory

Lightning-fast batch pre-translation with fuzzy matching and TMX import/export.

  • Batch pre-translation (30x faster)
  • Fuzzy + exact matching
  • TMX format support
🎯

2-Layer Prompt Architecture

System Prompts (auto-selected by mode) + Unified Prompt Library with folders, favorites, and multi-attach.

  • Layer 1: System Prompts (CAT tags, auto-selected)
  • Layer 2: Prompt Library (unlimited folders)
  • AI Assistant for prompt generation
📊

Bilingual Review Interface

Grid, List, and Document views for reviewing AI translations. Professional CAT tool interface.

  • Status tracking (Draft/Reviewed/Confirmed/Locked)
  • Multi-selection (Ctrl/Shift)
  • Grid pagination for large projects
🔄

CAT Tool Integration

Import/export bilingual tables from memoQ, CafeTran, and Trados Studio.

  • memoQ DOCX support
  • CafeTran pipe formatting
  • Trados bilingual tables
🌐

Superbrowser

Multi-chat AI browser with ChatGPT, Claude, and Gemini side-by-side in one window.

  • Three-column layout with resizable dividers
  • Persistent sessions — stay logged in
  • Custom URL support
📄

PDF Rescue (AI OCR)

Transform badly-formatted PDFs into clean, editable documents with GPT-4 Vision OCR.

  • One-click PDF import
  • Smart redaction/stamp handling
  • Professional DOCX export
