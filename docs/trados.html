Features

Professional terminology and AI tools inside Trados Studio

🔎

TermLens

Inline terminology display that shows source text word by word with glossary translations directly underneath each matched term.

  • Color-coded chips: blue, pink (project), yellow (non-translatable), green (MultiTerm)
  • Click or Alt+1–9 to insert terms into target
  • Right-click to edit, delete, or mark terms
  • Term Picker (Ctrl+Shift+G) for segments with many matches
  • Quick-add shortcuts: Alt+Down, Alt+Up, Ctrl+Alt+T
🤖

AI Assistant

Project-aware conversational AI assistant in a separate dockable panel.

  • Access to current segment, termbase terms, and TM matches
  • Multimodal: paste or drag-drop images
  • OpenAI, Anthropic, Google, and Ollama providers
  • Apply responses directly to the active segment
  • 14 built-in domain prompts
⚡

Batch Translate

AI-powered translation of multiple segments at once with full prompt control.

  • Translate selected segments, untranslated, or entire document
  • Choose provider, model, and prompt per batch
  • Progress tracking with real-time log
  • Respects existing translations (skip confirmed)
📗

MultiTerm Support

Reads MultiTerm .sdltb termbases attached to your Trados project and displays their terms as green chips.

  • Automatic detection from project settings
  • Green chips alongside Supervertaler terms
  • Auto-refresh when terms change in MultiTerm
  • Tooltip with all translations and metadata

Installation

Get up and running in under a minute

1

Download

Download the latest .sdlplugin file from the GitHub Releases page.

2

Install

Double-click the .sdlplugin file. Trados will install it automatically.

3

Restart & Use

Restart Trados Studio. The Supervertaler panel appears in the Editor view.
